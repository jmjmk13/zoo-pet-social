# Zoo - 宠物垂直领域社交平台 Demo

## 简介

Zoo 是一款专为宠物主人设计的社交平台，通过"以宠物为媒介"的创新社交方式，帮助社恐人群降低社交门槛，轻松结识附近的狗友猫友。

## 功能特点

- **宠物档案创建** - 为你的宠物创建个性化档案
- **附近宠物发现** - 基于距离展示附近的宠物主人
- **约遛弯功能** - 发起或参加遛弯邀请
- **动态分享** - 分享宠物日常，互动点赞评论

## 如何运行

### 方式一：直接打开（推荐）

双击 `zoo-demo.html` 文件，在浏览器中直接打开即可体验。

### 方式二：本地服务器

如果你已经启动了本地服务器（如 Python 的 http.server），可以通过以下方式访问：

```bash
# 在项目目录下启动服务器
python -m http.server 8080

# 然后在浏览器中访问
http://localhost:8080/zoo-demo.html
```

## 体验指南

1. **首次使用**：系统会引导你创建宠物档案，包括昵称、种类、品种、年龄和性格特点
2. **首页动态**：查看其他宠物主人分享的日常动态
3. **发现附近**：浏览附近的宠物主人，按类型筛选，添加好友
4. **约遛弯**：查看遛弯邀请或发起自己的邀请
5. **个人中心**：管理宠物档案，查看统计数据

## 技术栈

- HTML5 + CSS3 + JavaScript（纯前端实现）
- 无需后端，使用 localStorage 存储数据
- 移动端优先的响应式设计
- 极简美观的 UI 风格
